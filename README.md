# FYS4411 Project 1 report package

This package contains a compiled Project 1 report that uses the single-column figures in `report/figures/` and the LaTeX table snippets in `report/tables/`.

The report is written for a two-column layout. Each figure is included at `\columnwidth`, and every figure and table is referenced in the text.

To compile locally:

```bash
cd report
pdflatex project1_report.tex
pdflatex project1_report.tex
```
