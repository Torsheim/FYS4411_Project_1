# Selected Project 1 result files

These CSV files are the selected numerical results used to generate the report figures and tables. They are intentionally committed because the Project 1 instructions ask for selected results in the GitHub repository.

Main files:

- `project1_b_bruteforce.csv`: brute-force Metropolis scan for the non-interacting oscillator.
- `analytic_vs_numerical_energy.csv`: analytic versus finite-difference local-energy timing comparison.
- `project1_c_importance.csv`: importance-sampling time-step scan at the exact parameter `alpha=0.5`.
- `project1_c_importance_alpha045.csv`: finite-variance importance-sampling time-step scan at `alpha=0.45`.
- `project1_d_optimization.csv`: gradient optimization of `alpha`.
- `project1_e_blocking_alpha045.csv` and `project1_e_summary_alpha045.csv`: blocking/bootstrap analysis.
- `project1_g_interacting.csv`: interacting hard-core energy scan for `N=10,50,100`.
- `project1_h_radial_density_no_jastrow.csv` and `project1_h_radial_density_jastrow.csv`: radial one-body densities.
- `project1_h_pair_distance.csv` and `project1_h_pair_distance_summary.csv`: pair-distance diagnostic with and without Jastrow.
